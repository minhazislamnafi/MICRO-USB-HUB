# Specification

* USB HUB
* 1 upstream & 4 downstream
* 4 USB type-C (1 upstream & 3 downstream) and 1 USB type-A
* Chip- Fe1.1s SSOP-28
* dimentions: x x (LxWxH)

# Why you will need this micro usb hub?
Beacuse you definitly need a type-C usb hub if your using a desktop computer. also as its name suggests its very small and compact measures only (50x25x16)mm. So you should build one;)

# Documentation
In this repository, you can find 3D printable STL files file the USB hub enclosure, Garber, and all the other
files to build your own 4-port compact USB HUB.
